DESCRIPTION
***************************************************************************************
Write a program to classify integers as perfect, abundant, or deficient.  A positive
integer, n, is said to be perfect if the sum of its proper divisors equals the integer
itself.  Proper divisors include 1, but not the number itself.  If the sum of the
proper divisors is less than n, the integer is deficient.  If the sum of the proper
divisors is greater than n, the integer is abundant.


INPUT SPECIFICATION
***************************************************************************************
The input file "input.txt" contains a list of integers to be processed.  The first
line in the file is the number of integers to process.


OUTPUT SPECIFICATION
***************************************************************************************
Output a file named "output.txt" which exactly matches the file "sample-output.txt".
