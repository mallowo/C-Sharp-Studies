DESCRIPTION
*********************************************************************************
Write a program to convert a phrase into a sequence of binary numbers.


INPUT SPECIFICATION
*********************************************************************************
The file "input.txt" contains the phrase to be converted to binary.


OUTPUT SPECIFICATION
*********************************************************************************
Output a file called "output.txt" containing the binary sequence. Your output
file should exactly match "sample-output.txt".


PERMITTED .NET CLASSES
*********************************************************************************
System.IO.StreamReader
System.IO.StreamWriter
